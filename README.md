# MLP.simplify()

A simplified, student-friendly breakdown of Machine Learning concepts,
created from lecture notebooks and classroom discussions.

## What is this?
This repository contains weekly Markdown notes and FAQs derived from
Google Colab notebooks used during the Machine Learning course.

Each week focuses on:
- Core concepts
- Common student doubts
- Code intuition
- Exam-oriented clarifications

## Structure
weeks/   → Weekly markdown explanations  
assets/  → Images, plots, diagrams  

## Goal
To make Machine Learning clear, intuitive, and approachable.
